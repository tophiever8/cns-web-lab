# cns-web-lab

## 项目简介

cns-web-lab 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅用于存放各类前端页面示例、实验性项目、文档页面或静态内容，方便统一管理和版本控制。

## 目录结构

```
cns-web-lab/
├── index.html          # 仓库首页 / 导航页
├── pages/              # 存放独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

## 页面归档说明

- 所有独立 HTML 页面均放置在 `pages/` 目录下，每个页面为一个完整可访问的 HTML 文件。
- 页面之间相互独立，不共享状态或数据，可直接通过浏览器打开查看。
- 如需引用公共资源（如样式、脚本），请统一放置于 `assets/` 目录中，并在页面内以相对路径引用。
- 页面命名建议使用小写英文字母、数字和连字符（`-`），避免使用特殊字符。

## 维护说明

- 本仓库由维护者不定期更新，可包含新增页面、修正错误或优化现有内容。
- 欢迎提交 Issue 或 Pull Request 来建议新增页面或报告问题。
- 所有贡献请遵循简洁、清晰、独立的原则，避免引入外部依赖或复杂构建流程。
- 请勿提交与仓库无关或包含敏感信息的文件。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器直接打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 或其他静态托管服务部署本仓库，使页面可在线访问。
